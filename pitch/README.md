# Pitch

This folder contains the series pitch and series bible for Droidscouts.

## Contents

| Path | Description |
|---|---|
| `series-bible.md` | Canonical high-level document covering premise, world, cast, and tone |
| `pitch-deck/` | Pitch presentation materials (Markdown or exported formats) |
| `loglines.md` | One-line and one-paragraph series loglines, versioned for different audiences |

## File Format

All text assets are UTF-8 Markdown. Exported formats (PDF, etc.) may also live here but must have a Markdown source counterpart committed alongside them.

## Series Bible vs. World Folder

- The **series bible** (`pitch/series-bible.md`) is the outward-facing summary — premise, tone, audience, and cast snapshot. It is written for someone encountering the project for the first time.
- The **world folder** (`world/`) is the inward-facing canon reference — granular lore, faction details, and technology specs used during production.

When the series bible and world documents diverge, the world folder is authoritative for production decisions. Update the series bible to reflect locked decisions after major story milestones.
