# Art Direction

This folder contains all visual reference and art direction material for Droidscouts.

## Contents

| Path | Description |
|---|---|
| `style-guide/` | Overall visual tone, color palette, line weight, and panel style notes |
| `characters/` | Per-character design reference (appearance, costume, expressions) |
| `environments/` | Location and environment visual reference |
| `technology/` | Design reference for the lab, gear, robots, and other tech assets |
| `templates/` | Starter template for art direction notes |

## File Format

- Written reference and notes: UTF-8 Markdown (`.md`)
- Design images and scans (from Google Drive): place in the appropriate subfolder alongside a companion `.md` file describing what the image shows and any relevant production notes
- Do not commit large binary files without first checking `.gitignore` coverage

## Naming Conventions

- Image companion files: same base name as the image file (e.g., `helen-concept-v1.png` → `helen-concept-v1.md`)
- Written notes: descriptive lowercase hyphenated names (e.g., `style-guide/color-palette.md`)

## Panel Style Notes

Document panel layout conventions here once established — grid structure, bleed policy, caption box style, and SFX treatment. These notes are the reference that keeps art consistent across chapters and artists.
