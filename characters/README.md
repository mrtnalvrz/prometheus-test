# Characters

This folder contains all character reference material for Droidscouts.

## Contents

| Path | Description |
|---|---|
| `profiles/` | Individual character profile files |
| `relationships/` | Relationship maps and group dynamics documents |
| `templates/` | Starter template for new character profiles |

## Naming Conventions

- Character profiles: `profiles/firstname-lastname.md` (lowercase, hyphenated). For characters with one name or a codename, use that name.
- Use the character profile template in `templates/character-profile-template.md` for all new profiles.

## File Format

All text assets are UTF-8 Markdown. See `docs/creative_asset_formats.md` for the canonical character profile template once that document is created (Task 2 in `Final-Productization-Checklist.md`).
