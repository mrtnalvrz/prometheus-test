3. EPISODE SCRIPT PDF-TO-MARKDOWN CONVERSION (Checklist Task 2 — CLOSED)
================================================================================

Written: scripts/pdf_to_markdown.py
  - Extracts text from PDFs using pdfplumber (installed as dependency)
  - Uses x-coordinate analysis to classify screenplay elements:
      x0 < 160    → action lines / sluglines
      x0 160-220  → dialogue / parentheticals
      x0 220-290  → character names (centered)
      x0 > 380    → transitions / page numbers
  - Fixes encoding artifacts: U+FFFD replacement chars → straight apostrophes
  - Fixes split contractions (e.g. "there' s" → "there's")
  - De-duplicates doubled characters from PDF rendering artifacts
  - Removes page number markers (e.g. "22..")
  - Applies Markdown formatting: bold sluglines, bold character names,
    italic parentheticals, right-aligned transitions

Converted all four episode PDFs:

  create/scripts/ch01-script.md   — 1,107 lines | 23 sluglines | 137 char cues
