# Season 1

This folder contains all creative assets scoped to Season 1 of Droidscouts (10–12 chapters).

## Contents

| Path | Description |
|---|---|
| `chapter-tracker.md` | Master status table for all Season 1 chapters — title, synopsis, status |
| `outlines/` | Chapter-level plot outlines (one file per chapter) |
| `scripts/` | Full panel scripts (one file per chapter) |
| `templates/` | Starter templates for outlines and scripts |

## Naming Conventions

- Chapter outlines: `outlines/ch01-outline.md`, `ch02-outline.md`, etc.
- Chapter scripts: `scripts/ch01-script.md`, `ch02-script.md`, etc.
- Use zero-padded two-digit chapter numbers so files sort correctly.

## File Format

All text assets are UTF-8 Markdown. See `docs/creative_asset_formats.md` for the canonical template structures once that document is created (Task 2 in `Final-Productization-Checklist.md`).

## Status Definitions (used in `chapter-tracker.md`)

| Status | Meaning |
|---|---|
| `concept` | Chapter exists as an idea only — no written outline |
| `outline` | Plot outline drafted |
| `draft` | Panel script in progress |
| `complete` | Script finalized and ready for art |
